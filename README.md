# Pick Lab 対戦・選出シミュレーター（MVP）
- 3vs3 / 見せ合い無し / 交代はターン消費 / 状態異常なし / OHKO(命中30%)
- ブラウザだけで動くCPU対戦MVP。PvPは後でサーバー化する前提。

## 起動
ローカルで index.html を直接開くと fetch が制限される場合があります。

### Python
```bash
python -m http.server 8000
```
http://localhost:8000
